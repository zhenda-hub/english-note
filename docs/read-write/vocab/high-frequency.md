
## 可能性的单词

- absolutely / definitely
- probably
- maybe
- unlikely

## 年份

年份可以拆分成两个 两位数来说:

- 1980: nighteen eighty
- 2008: twenty eight
- 2022: twenty twenty two

## 时间

a quarter to nine => 8:45
a quarter past nine => 9:15

- “past” → 过了某个小时
- “to” → 差到下一个小时

## 大数字

| 中文单位 | 数字    | 英文单位            |
| -------- | ------- | ------------------- |
| 千       | 10³     | **thousand**        |
| **万**   | 10⁴     | ten thousand        |
| **百万** | **10⁶** | **one million**     |
| 千万     | 10⁷     | ten million         |
| **亿**   | **10⁸** | one hundred million |
| 十亿     | 10⁹     | **one billion**     |
| 百亿     | 10¹⁰    | ten billion         |
| 千亿     | 10¹¹    | hundred billion     |
| 万亿     | 10¹²    | one trillion        |
| 千万亿   | 10¹⁵    | one quadrillion     |
