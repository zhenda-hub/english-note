## 年份

年份可以拆分成两个 两位数来说:

- 1980: nighteen eighty
- 2008: twenty eight
- 2022: twenty twenty two

## 星期

## 时间

a quarter to nine => 8:45
a quarter past nine => 9:15

- “past” → 过了某个小时
- “to” → 差到下一个小时

## 小数字

| 时间单位 | 符号 | 英文名称    | 与秒的换算             | 典型参照物 / Typical Reference                                               |
| :------- | :--- | :---------- | :--------------------- | :--------------------------------------------------------------------------- |
| 毫秒     | ms   | Millisecond | 千分之一秒 (10⁻³)      | 蜂鸟振翅一次 / A hummingbird flaps its wings once. (~5ms)                    |
| 微秒     | μs   | Microsecond | 百万分之一秒 (10⁻⁶)    | SSD随机读取 / An SSD random read. (~50-150μs)                                |
| 纳秒     | ns   | Nanosecond  | 十亿分之一秒 (10⁻⁹)    | CPU一个时钟周期 / One clock cycle of a CPU (e.g., ~0.3ns for 3 GHz)          |
| 皮秒     | ps   | Picosecond  | 万亿分之一秒 (10⁻¹²)   | 光传播0.3毫米 / Light travels 0.3 millimeters.                               |
| 飞秒     | fs   | Femtosecond | 千万亿分之一秒 (10⁻¹⁵) | 分子振动周期 / Period of a molecule's vibration.                             |
| 阿秒     | as   | Attosecond  | 百亿亿分之一秒 (10⁻¹⁸) | 电子绕原子核运动的时间尺度 / Time scale of electron motion around a nucleus. |
