## 重音与节奏（比发音更重要）

重读和轻读是英语发音的核心特点

英语是重音节奏语言， 其节奏由重读音节主导

特点：

- 重读音节之间的间隔时间相对固定。
- 重读音节拉长、清晰
- 非重读音节压缩、弱化


| 类型  | 是否重读 |
| --- | ---- |
| 名词  | ✅    |
| 动词  | ✅    |
| 形容词 | ✅    |
| 介词  | ❌    |
| 冠词  | ❌    |


## 语音难点

英语语音的核心目标只有一个：追求“省力”

于是就产生了：
- 连读
- 弱读
- 失去爆破
- 同化
- 缩读


### 连读（Linking）——单词“黏”在一起

最常见的情况：一个词以辅音结尾，下一个词以元音开头，就直接连起来读。
常见例子：

pick it up → 听起来像 “pickitup”
not at all → “notatall”
hold on → “holdon”

另一个超常见：what you → “whatcha” （what + you 连读成 /wɒtʃə/）
多听美剧，你会发现几乎每句都有连读，不连会听起来很僵硬。


### 弱读

弱读是口语最明显的特征！不弱读，听起来像机器人。

英语里很多功能词（如and, to, for, can, the等）在句子中不重读时，会弱化成很短的音，通常是 /ə/（类似“呃”）或 /ɪ/。

最常用的弱读词（记住这几个就行）：


| 词  | 正常读（强读） | 弱读（口语中） | 例子                            |
| :-- | :------------- | :------------- | :------------------------------ |
| and | /ænd/          | /ən/ 或 /n/    | fish and chips → “fish'n chips” |
| to  | /tuː/          | /tə/           | go to bed → “go tə bed”         |
| for | /fɔːr/         | /fər/          | wait for me → “wait fer me”     |
| can | /kæn/          | /kən/          | I can go → “I kən go”           |
| of  | /ɒv/           | /əv/           | a cup of tea → “cupəv tea”      |
| the | /ðiː/          | /ðə/           | the apple → “ðə apple”          |
| you | /juː/          | /jə/           | Thank you → “Thank yə”          |



### 失去爆破


爆破音 /p, b, t, d, k, g/ 在某些位置只形成口型，不完全爆破（持阻但不除阻）。
规则

两个爆破音相邻：前一个失去爆破。
示例：good ~ boy → /ɡʊb bɔɪ/（/d/ 不爆破）
hot ~ coffee → /hɒk kɒfi/（/t/ 不爆破）

爆破音 + 其他辅音：常失去爆破，尤其是 /t, d/。
示例：next ~ please → /neks pliːz/（/t/ 失去）
old ~ man → /əʊl mæn/（/d/ 失去）

不完全爆破：有时有轻微气流，但不明显。



## 误区

**过度纠结“完美发音”**：语音的核心是“清晰易懂”，而非和母语者完全一致，避免因追求单个音的完美而放弃流畅度；
